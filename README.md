# 🏥 Hospital Management System

A console-based Hospital Management System developed in Java that helps manage hospital operations efficiently. The system allows users to add patient records, view patients and doctors, and book appointments through a simple menu-driven interface.

## ✨ Features

- 👤 Add Patient
- 📋 View Patients
- 👨‍⚕️ View Doctors
- 📅 Book Appointment
- 🚪 Exit Application

## 🛠️ Technologies Used

- Java
- JDBC
- MySQL
- IntelliJ IDEA
- Git & GitHub

## ⚙️ Prerequisites

- Java JDK 8 or later
- MySQL Server
- MySQL JDBC Driver
- IntelliJ IDEA

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/Prakhar140/Hospital-Management-System.git
```

2. Open the project in IntelliJ IDEA.

3. Create the required MySQL database and tables.

4. Update the database credentials in the project.

5. Run the application.

## 💻 How It Works

After starting the application, the following menu is displayed:

```
1. Add Patient
2. View Patients
3. View Doctors
4. Book Appointment
5. Exit
```

Choose an option by entering the corresponding number.
